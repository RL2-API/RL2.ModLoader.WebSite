- GET /api/v2/mod-list/{page}
  • Returns a paginated list of basic mod info.
  • The page parameter is optional, and by default is 0.
  • Accepts an optional JSON payload with the following schema: 
    {
    "mods_per_page"  : u32        [default = 10]
}
  • The return value JSON schema: 
    // Success
{
    "status" : u16  [default = 200]
    "page"   : u32
    "data"   : list of {
        "name"         : string
        "author"       : string
        "icon_src"     : string or null
        "short_desc"   : string
    }
}

// Failure
{
    "status" : u16
    "error"  : string
}
