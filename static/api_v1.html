- GET /api/v1/mod-list:
  • Returns a JSON list of all mods from the database, or {{ axum::http::StatusCode::INTERNAL_SERVER_ERROR }}
  • JSON schema for every entry: 
    {
    "name"       : string
    "author"     : string
    "icon_src"   : string or null
    "short_desc" : string
}
- GET /api/v1/mod/{name}:
  • Returns a JSON object representing info about the mod with the specified name
  • JSON schema of the return value 
    {
    "mod_info" : {
        "name"      : string
        "author"    : string
        "icon_src"  : string or null
        "long_desc" : string (markdown)
    }
    versions: list of {
        "link"      : string (download file URL)
        "version"   : string
        "changelog" : string (markdown)
    }
}
