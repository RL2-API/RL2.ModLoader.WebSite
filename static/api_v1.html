GET /api/v1/mod-list

Returns a JSON list of all mods from the database. 
// 200: Data found
{
    "name"       : string
    "author"     : string
    "icon_src"   : string or null
    "short_desc" : string
}

GET /api/v1/mod/{name}

Returns a JSON object representing info about the mod with the specified name 
// 200: Data found
{
    "mod_info" : {
        "name"      : string
        "author"    : string
        "icon_src"  : string or null
        "long_desc" : string (markdown)
    }
    versions: [
        {
            "link"      : string (download file URL)
            "version"   : string
            "changelog" : string (markdown)
        }
    ]
}

// 404: Page not found
{  }
