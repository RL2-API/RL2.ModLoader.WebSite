- GET /api/mod-list:
  • Returns a JSON list of all mods from the database, or {{ axum::http::StatusCode::INTERNAL_SERVER_ERROR }}
  • JSON schema for every entry: 
    {
    "name"       : string,
    "author"     : string,
    "icon_src"   : string,
    "short_desc" : string
}
- GET /api/{name}:
  • returns a JSON object representing info about the mod with the specified name
